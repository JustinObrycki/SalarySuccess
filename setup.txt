The following npm modules must be added before running database.js:
npm install express
npm install mongodb
npm install bcrypt
npm install prompt-sync
npm install node-rsa
npm install multer


npx browserify salary_negotiation_db_v1.js -o sal_alg.js


Once complete "node .\database.js" can be run to start web application.
